# Programming_Fundamentals_Final_Project
* **Final Project**
  * Final project for my first semester Programming Fundamentals course.
* **Features of this Project**
  * This project is a Student Grading System written in C++. 
  * It allows a user to manage student records by adding, updating, deleting, and displaying student information. 
  * Each student has a name, roll number, and marks for five subjects. 
  * The program automatically calculates the total marks, average percentage, and assigns a grade (A, B, C, or D) along with a personalized feedback message. 
  * All data is saved to and loaded from a file (students.txt) so records persist between runs. Additional features include viewing a summary report (class average, topper, grade distribution) and printing an individual student’s result.
* **Lab Tasks**
  * Task # 1: Time conversion from hours, minutes and seconds to seconds
  * Task # 2: Time conversion from seconds to hours, minutes and seconds
